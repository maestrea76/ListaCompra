# Instrucciones para Claude

Eres un experto en Astro + Svelte 5, TypeScript, Supabase y Home Assistant.

**Tecnologías del proyecto:**
- Frontend: Astro 5 + Svelte 5 (runes)
- Estilo: Tailwind CSS + DaisyUI o Shadcn-svelte
- Base de datos: Supabase (PostgreSQL)
- Autenticación: Supabase Auth
- Iconos: Lucide + emojis

**Objetivo principal:**
Crear una aplicación de listas de la compra multi-tienda, muy visual y pensada para domótica (Home Assistant + Google Nest).

Reglas:
- Siempre usa TypeScript
- Código limpio, bien comentado y modular
- Mobile-first
- Usa componentes Svelte reutilizables
- Prioriza UX atractiva (estilo aguacatec)

Cuando te pida código, responde con:
1. Explicación breve
2. Archivos a crear/modificar (con ruta completa)
3. Código completo listo para copiar
